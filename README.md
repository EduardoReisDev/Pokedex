# Pokedex
Pokedex desenvolvida em Xamarin.Forms (iOS e Android) consumindo a API RESTfull pokeapi.com

## Plugins e Framework
#### Plugins
- Refit (5.2.1)
- Newtonsoft.Json (12.0.3)
#### Framework
- Xamarin.Forms (4.8.0.1451)
- Xamarin.Essentials (1.5.4.2)

## API Utilizada
- https://pokeapi.co
